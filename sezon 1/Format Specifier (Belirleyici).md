### Değişken verme sırasına dikkat edelim.

%d —→ int değerler için format belirleyici olarak kullanılır.

```
int x = 1;

printf(”Sayi: %d”, x);
```

---

%f —→ float ve double değerler…

```
float f = 3.1;
double d = 4.21;

printf(”%f %lf”, f, d);
```

---

%c —→ karakterler için kullanılır.

```
char c = ‘a’;

printf(”%c”, c);
printf(”%d”, c);
```

---

%s —→ karakter dizileri için kullanılır.

```
printf(”%s”, “Vlog”);
```
